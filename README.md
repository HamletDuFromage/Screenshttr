# Screenshttr

## Features

- Take a screenshot of the current
- Take a screenshot of each open tab

For single screenshots, I recommend using Firefox's built-in screenshot too.

## How to build
```
$ npm install --save-dev
$ npm run build
```
The resulting extension will be in `dist/`.
